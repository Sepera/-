Задача. 
Перемещающиеся по городу объекты в произвольные моменты времени передают по каналу связи сигналы о своём местоположении.
Входные данные представляют собой следующие поля с разделителями ";":
- Дата и время (формат DD.MM.YYYY hh:mm);
- Имя объекта (формат текст);
- Широта (формат в градусах в виде десятичной дроби);
- Долгота (формат в градусах в виде десятичной дроби).

Необходимо написать программу, которая:
1) читает указанную базу данных (файл);
2) считает и визуализирует статистику по данным:
  2.1) Для каждого объекта подсчитывается количество сигналов, поступивших от него в течение каждых суток. 
       Визуализируется это с помощью графика, где по оси абсцисс откладываются дни (т.е. сутки), а по оси ординат - количество сигналов.
       Для каждого объекта должен быть сделан вывод в виде текста, в какой день у него зарегистрирована максимальная активность.
  2.2) Для каждого объекта подсчитывается количество сигналов, поступивших от него в определённные часы суток. Т.е. с 00:00 по 00:59 в сумме за все дни, с 01:00 по 01:59 в сумме за все дни, и т.д.
       Визуализируется это с помощью графика, где по оси абсцисс откладываются часы (24 значения), а по оси ординат - количество сигналов.
	   Должен быть сделан вывод в виде текста, какой объект более активен ночью, какой утром, какой днём, какой вечером.
	   Для простоты разделим сутки так: ночь с 00:00 до 05:59, утро с 06:00 до 11:59, день с 12:00 до 17:59, вечер с 18:00 до 23:59.
3) делает поиск ситуаций, когда объекты подходили достаточно близко друг к другу. Т.е. необходимо найти все пары сигналов от разных объектов, отличающиеся на время не большее T, а расстояние между ними должно быть не больше D.
   Параметры T (в минутах) и D (в метрах) должны задаваться пользователем перед началом анализа данных.
